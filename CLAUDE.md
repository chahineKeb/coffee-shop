## Project Overview

This is a school evaluation project: a landing page for a fictional coffee shop. The stack is freely chosen by the student. The project does not yet have a chosen framework — if one is initialized, update this file accordingly.

## Assignment Requirements

**Required sections on the page:**
- Hero: café name, tagline, CTA button
- Menu/offerings: drinks, formulas, specialties
- Ambiance/values: story, coffee origins, spirit of the place
- Secondary CTA: reservation, newsletter, or location
- Footer: fictional address, social media links

**Technical constraints:**
- Responsive: mobile, tablet, desktop
- SEO fundamentals (semantic HTML, meta tags, titles, alt text)

**Deliverables:**
- Public GitHub repo
- Deployed on Netlify or Vercel
- Submit both links on Discord
